# Extension Icons

This directory should contain the extension icons in the following sizes:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

## Creating Icons

You can create simple placeholder icons using any image editor or online tool.

For now, you can:
1. Use an online icon generator like https://www.favicon-generator.org/
2. Create simple colored squares as placeholders
3. Use a LinkedIn-themed icon (blue background with "LI" text)

## Quick SVG Placeholder (can be converted to PNG)

```svg
<svg width="128" height="128" xmlns="http://www.w3.org/2000/svg">
  <rect width="128" height="128" fill="#0A66C2"/>
  <text x="64" y="80" font-family="Arial" font-size="60" fill="white" text-anchor="middle" font-weight="bold">LI</text>
  <text x="64" y="105" font-family="Arial" font-size="18" fill="white" text-anchor="middle">Auto</text>
</svg>
```

Save this as icon.svg and convert to PNG at different sizes using an online converter.
